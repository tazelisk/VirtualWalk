Linderung von chronischen neuropatischen Schmerzen Videosystem basierend auf Spiegeltherapie

JETZT SPENDEN

Overview

Chronische,neuropathische Schmerzen nach Rückenmarksverletzungen sind nicht gut verstanden und schwer zu behandeln.

Eine mögliche Ursache der Schmerzen ist eine Nichtübereinstimmung zwischen motorischen Befehlen und sensorischen Rückmeldungen.

Um die chronischen Schmerzen zu behandeln, wird oft eine visuelle Therapierungsmethode angewendet. Eine solche Variante stellt das Virtual Walk System dar. Die Therapierung durch dieses System beruht auf einer visuellen Ausgabe auf einem TV-Bildschirm, die durch den Paraplegiker wahrgenommen wird.
